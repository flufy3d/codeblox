## 🎯 今天要做什么

把上节那排格子，刷成**五颜六色的彩虹跑道**！而且每次试玩颜色都不一样——这就靠**随机数**。

## ✨ 新知识：math.random 是“随机抽一个”

`math.random`（随机数）就像**掷骰子**：每次都给你一个说不准的数。

> `math.random(1, 255)` = “在 1 和 255 之间，随便抽一个整数。”

颜色用 `Color3.fromRGB(红, 绿, 蓝)` 表示，三个数都在 **0~255**。我们让红、绿、蓝都随机抽——混出来就是**随机的颜色**，每个格子都不一样！

<StudioStep n={1} title="新建一个服务器脚本">
  在 Explorer 里右键 **ServerScriptService → Insert Object → Script**。
</StudioStep>

<StudioStep n={2} title="整段抄入彩虹跑道代码">
  整段换成下面这些。先原样跑：

```lua
for i = 1, 10 do
	local block = Instance.new("Part")
	block.Size = Vector3.new(4, 1, 4)
	block.Position = Vector3.new(i * 5, 5, 0)
	block.Anchored = true
	block.Material = Enum.Material.Neon
	block.Color = Color3.fromRGB(math.random(1, 255), math.random(1, 255), math.random(1, 255))
	block.Parent = workspace
end
```

  和上节一样生成 10 个格子，但这次每个格子的 **Color** 是随机抽的，还把 **Material** 设成 **Neon（霓虹）** 让它发光。
</StudioStep>

<StudioStep n={3} title="试玩，看彩虹亮起来">
  点 ▶️ **Play**。一排**发光的彩色格子**出现了——你的彩虹跑道！🌈
</StudioStep>

<StudioStep n={4} title="再玩几次，每次都不同">
  ⏹️ **Stop**，再 ▶️ **Play**。注意看：颜色组合**和上次不一样**！因为每次都是随机抽的。
</StudioStep>

<Callout type="tip">
  三个 `math.random(1, 255)` 分别管红、绿、蓝三种光。它们各自随机，混在一起，就成了千变万化的颜色。
</Callout>

## 🕹️ 看看你的成果

一条**会发光、每次都不同**的彩虹跑道！这就是本单元的核心成果之一，最后一课我们会让它动起来。🌈

<Callout type="warning">
  如果格子全是黑的：检查 `Color3.fromRGB(...)` 里有没有写三个 `math.random(1, 255)`，三个数都要在 0~255 之间。
</Callout>
