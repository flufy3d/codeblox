## 🎯 今天要做什么

往计分卡里放一个**分数**！这一节做完，游戏**左上角**就会真的出现你的分数，像真正的游戏一样。

## ✨ 新知识：IntValue 是一个“数字盒子”

上节课我们做好了空的 **leaderstats** 文件夹。现在往里放一个能装**整数**的盒子——**IntValue（整数值）**，用它来记分数。

我们还是用 `Instance.new` 来造：

```lua
local points = Instance.new("IntValue")
points.Name = "Points"
points.Value = 0
points.Parent = leaderstats
```

- `Name = "Points"`：盒子的名字（左上角就显示这个名字）
- `Value = 0`：一开始是 0 分
- `Parent = leaderstats`：把盒子放进计分卡文件夹里

把它接到上节课的代码后面，就是完整的计分板脚本：

```lua
local Players = game:GetService("Players")

Players.PlayerAdded:Connect(function(player)
	local leaderstats = Instance.new("Folder")
	leaderstats.Name = "leaderstats"   -- 名字必须正好是 leaderstats（全小写）
	leaderstats.Parent = player

	local points = Instance.new("IntValue")
	points.Name = "Points"
	points.Value = 0
	points.Parent = leaderstats
end)
```

<Callout type="info">
  这段依然放在 **ServerScriptService** 的 Script 里。它对每个加入的玩家都发一张带 **Points** 的计分卡。
</Callout>

## 🕹️ 看看你的成果

左上角亮出了 `Points 0`。计分板成型了！只是现在分数永远是 0，下一节我们就让“**碰到我的那个玩家**”能被认出来，为加分做准备。

<Callout type="warning">
  左上角没出现分数？检查名字是不是全小写 `leaderstats`，以及分数项的 `Parent` 是不是指向 `leaderstats`。
</Callout>
