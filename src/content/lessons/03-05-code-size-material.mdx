## 🎯 今天要做什么

上节课你用代码改了颜色。今天再学两招：用代码把方块**变大**，还能让它**发光**（Neon 霓虹）！🪄

## ✨ 新知识：还能改大小和材质

给方块下命令的写法都很像，左边是“方块.要改的东西”，右边是“设成什么”。

**改大小**用 `Size`，要给三个数字（长、宽、高）：

```lua
part.Size = Vector3.new(8, 8, 8)
```

**改材质**用 `Material`，把它设成 Neon 就会发光：

```lua
part.Material = Enum.Material.Neon
```

`Vector3` 和 `Enum.Material.Neon` 看起来有点长，先**照抄**就好，会用就行！

<StudioStep n={1} title="把命令加到脚本里">
  打开 MyPart 里的脚本，把它补成下面这样（在改颜色后面，再加改大小和材质两行）：

```lua
local part = script.Parent
part.BrickColor = BrickColor.new("Bright red")
part.Size = Vector3.new(8, 8, 8)
part.Material = Enum.Material.Neon
```
</StudioStep>

<StudioStep n={2} title="试玩看效果">
  点 ▶️ Play。方块会一下子**变大、变红、还开始发光**——三个命令一起生效了！
</StudioStep>

<StudioStep n={3} title="改改大小试试">
  把 `Vector3.new(8, 8, 8)` 里的数字换一换，越大方块越大。试试 `Vector3.new(4, 4, 4)` 或者 `Vector3.new(12, 1, 12)`（一块大薄板）。
</StudioStep>

<Callout type="tip">
  `Enum.Material.Neon` 里的 Neon 也能换别的材质，比如 `Enum.Material.Wood`（木头）、`Enum.Material.Grass`（草）。但 **Neon 最适合做我们这单元的发光方块**，先留着它！
</Callout>

## 🕹️ 看看你的成果

一个又大、又红、还会发光的方块——而且**大小、颜色、材质全是代码改的**！你已经能用一段脚本，一口气改好几样东西了。

<Callout type="warning">
  报错或没反应？检查这几点：① `Vector3` 和 `Material` 的大小写要对（V、M 大写）；② 三个数字之间用**英文逗号** `,`；③ `Neon` 第一个字母大写。下一课我们专门学怎么看懂报错。
</Callout>
