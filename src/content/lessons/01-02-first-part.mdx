## 🎯 今天要做什么

往你的世界里放进**第一个方块（Part）**，并在 **Explorer（资源管理器）** 里找到它。

## ✨ 新知识：Part 是积木，Explorer 是清单

- **Part（方块）** = 搭一切东西的“积木”。
- **Explorer（资源管理器）** = 世界里每样东西的“目录清单”；`Workspace` 是装“看得见的 3D 东西”的大盒子。

## 👣 跟我一步步做

1. 打开 **Explorer**：顶部 **View（视图）选项卡 → Explorer**。
2. 插入方块：**Home 选项卡 → Insert（插入）→ Part**。
   （也可以在 Explorer 里**右键 `Workspace` → Insert Object → Part**。）
3. 一个灰色方块出现在镜头中心。
4. 在 Explorer 里点开 `Workspace` 前的小箭头，找到刚出现的 `Part`，点它一下——它在 3D 里会高亮。
5. 按 **Ctrl + S** 保存，命名为“我的迷你乐园”。

## 🕹️ 看看你的成果

世界里有了一个方块，Explorer 的 `Workspace` 下能看到对应的 `Part`，点击它能选中。🎉
