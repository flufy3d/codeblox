## 🎯 今天要做什么

把你辛苦做出来的游戏，从“只存在这台电脑里”升级成“存在 Roblox 云端”。这样就算换台电脑、或电脑出了问题，你的作品也丢不了——它跟着你的账号走。

## ✨ 新知识：本地保存 vs 云端保存

想象你画了一幅画：

- **本地保存（Ctrl + S）** = 把画放进自己书桌的抽屉里。只有坐在这张书桌前才拿得到，抽屉坏了画也就没了。
- **云端保存（Save to Roblox）** = 把画寄存到一个超级安全的大仓库，凭你的账号随时随地都能取回来。

两个都要做：平时随手 Ctrl + S，做出阶段成果就 Save to Roblox 存一份到云端。

<Callout type="info">
  小提示：Roblox 对**不同年龄段**的账号，能不能公开发布、能不能社交，规定是不一样的。今天我们只做“存档到自己的云端”，还不涉及公开。具体能开哪些功能，以**家长账号控制台**和当时的**官方政策**为准——家长可以在控制台里管理孩子的发布与社交权限。
</Callout>

<StudioStep n={1} title="先本地存一次">
  按 **Ctrl + S**。如果是第一次，给文件起个名存到电脑上。这一步保证你眼前的进度先不丢。
</StudioStep>

<StudioStep n={2} title="打开 File 菜单">
  点左上角 **File（文件）选项卡 → Save to Roblox…（保存到 Roblox）**。
</StudioStep>

<StudioStep n={3} title="填名字和描述">
  在弹出的窗口里给作品起个**名字**（比如“我的障碍跑酷”），可以再写一句**描述**。这些只有你自己看得到。
</StudioStep>

<StudioStep n={4} title="点 Save 上传">
  点 **Save（保存）**，等进度条走完。完成后，这个作品就进了你账号的云端啦。
</StudioStep>

<StudioStep n={5} title="验证一下">
  关掉 Studio 再重新打开，在开始界面找 **My Games（我的游戏）**，应该能看到刚才那个名字。点开它，作品原封不动回来了！
</StudioStep>

## 🕹️ 看看你的成果

你的游戏现在有了两个“家”：一个在你电脑里，一个在 Roblox 云端。从今往后，重要进度记得两边都存。

<Callout type="warning">
  如果没成功？
  - **找不到 Save to Roblox**：确认你已经**登录了自己的 Roblox 账号**（右上角应显示用户名）；没登录就只能本地保存。
  - **My Games 里看不到**：可能上传还没传完，或登录的是另一个账号。等网络好一点，重新 File → Save to Roblox 再存一次。
</Callout>
