## 🎯 今天要做什么

把上一课学的“事件”真正用起来！让你的 `MagicFloor` 地板**被碰到时，就在 Output 喊一句话**。走上去试试，超有成就感！

## ✨ 新知识：把动作“连”到 Touched 上

我们要做三件事：

1. 先把方块存进盒子 `part`（第 2 课学过）。
2. 写好“碰到要做什么”——先简单地 `print` 一句话。
3. 用 `Connect` 把“碰到（Touched）”和这个动作**连起来**。

先整段抄下来，看懂结构：

```lua
local part = script.Parent

local function onTouch(hit)
	print("有人碰到我啦！")
end

part.Touched:Connect(onTouch)
```

- `local part = script.Parent`：抓住这块地板。
- `local function onTouch(hit)` … `end`：这是“碰到时要做的事”，先打印一句话。
- `part.Touched:Connect(onTouch)`：**碰到 → 就去做 onTouch**。这就是“发生某件事 → 就做某事”！

<Callout type="info">
  `hit` 是个变量，代表“碰到我的那个东西”（比如玩家的脚）。这节课先不用它，下一课会讲到。
</Callout>

你一踩上地板，Output 就喊出一句话——这是你第一次让“碰到”这件事**触发**了代码！下一课，我们把这句话换成真正的魔法：变色。✨

</Callout>
