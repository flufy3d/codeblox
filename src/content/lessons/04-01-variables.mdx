## 🎯 今天要做什么

学会用一个**盒子**把数字或文字“装起来”，需要的时候随时拿出来用。今天我们装一个分数，再用 `print` 把它显示出来！

## ✨ 新知识：变量是“贴标签的盒子”

想象你有很多小盒子，每个盒子贴一张标签写上名字，盒子里可以装东西。

在代码里，这种盒子叫**变量（variable）**：

- **标签** = 名字，比如 `score`（分数）
- **盒子里装的** = 值，比如 `0`

要做一个新盒子，前面写 `local`（意思是“我自己的盒子”）：

```lua
local score = 0
print(score)
```

第一行：做一个叫 `score` 的盒子，里面装 `0`。
第二行：把盒子里的东西拿出来 `print` 到 Output。

<Callout type="info">
  变量名只能用**英文**字母、数字、下划线（像 `score`、`my_points`），不能用中文。中文我们写在注释或讲解里就好。
</Callout>

<StudioStep n={1} title="插入一个脚本">
  在 Explorer 里**右键 Workspace → Insert Object → Script**。会出现一个写代码的窗口。
</StudioStep>

<StudioStep n={2} title="清空，写上这两行">
  把里面原来的字删掉，照着写：

```lua
local score = 0
print(score)
```
</StudioStep>

<StudioStep n={3} title="试玩看结果">
  打开 **View 选项卡 → Output**，点顶部 ▶️ **Play**。Output 里会出现一个 **0**——那就是盒子里的东西被拿出来了！
</StudioStep>

## 🕹️ 看看你的成果

Output 窗口里出现了 `0`，说明你的“盒子”成功装上了数字，还被拿了出来。试试把 `0` 改成 `100`，再玩一次，看看 Output 变成什么。✨

<Callout type="warning">
  如果 Output 里没东西：检查是不是忘了写 `print(score)` 这一行，或者 `score` 拼错了（大小写也要一样）。
</Callout>
