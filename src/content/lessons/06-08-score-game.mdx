## ✨ 新知识：把两份脚本组合成一个游戏

我们这个单元做了两样东西，现在把它们拼起来：

1. **计分板**（放在 **ServerScriptService**，给每个玩家发 `leaderstats` 和 `Points`）
2. **得分垫**（放在每块 **Pad** 里，碰到就加分，还带 debounce 防狂涨）

这就是完整的**计分板脚本**（在 ServerScriptService）：

```lua
local Players = game:GetService("Players")

Players.PlayerAdded:Connect(function(player)
	local leaderstats = Instance.new("Folder")
	leaderstats.Name = "leaderstats"   -- 名字必须正好是 leaderstats（全小写）
	leaderstats.Parent = player

	local points = Instance.new("IntValue")
	points.Name = "Points"
	points.Value = 0
	points.Parent = leaderstats
end)
```

这是放在**每一块 Pad 里**的**得分垫脚本**（带 debounce）：

```lua
local pad = script.Parent
local debounce = false

pad.Touched:Connect(function(hit)
	local player = game.Players:GetPlayerFromCharacter(hit.Parent)
	if player and not debounce then
		debounce = true
		player.leaderstats.Points.Value += 1
		task.wait(1)
		debounce = false
	end
end)
```

<Callout type="tip">
  好消息：**复制 Pad 的时候，里面的脚本会一起被复制！** 所以你做好一块 Pad，复制几份摆开，就有了一整排得分垫，每块都能加分。
</Callout>

<StudioStep n={1} title="先放好计分板">
  确认 **ServerScriptService** 里有那段计分板脚本（上面第一段）。它管左上角的 Points。
</StudioStep>

<StudioStep n={2} title="做第一块得分垫">
  在 **Workspace** 放一个 **Part**，里面 **Insert Object → Script**，写上第二段得分垫代码。Play 踩一下，确认能加分。
</StudioStep>

<StudioStep n={3} title="复制成一条收集路线">
  在 Explorer 里**右键这块 Pad → Duplicate（复制）**，复制 4-5 份，在地图上摆成一条路或一圈。脚本会跟着复制，每块都能加分。
</StudioStep>

<StudioStep n={4} title="开玩！">
  按 **▶️ Play**，沿着路线一块块踩过去——分数一路往上涨！
</StudioStep>
