## 🎯 今天要做什么

把这一单元学的**全部用上**，做出本单元的大成果：一块**踩上去就变色、还能消失让人穿过去**的《魔法地板》！做完——**叫家人来踩一脚秀一秀！** ✨

## ✨ 复习：三样法宝合体

今天不学新概念，把学过的串成一条龙：

> **变量**（存住方块）→ **Touched 事件**（碰到）→ **函数**（把要做的事打包）

我们就在 AUTHORING 那段经典结构上，往函数里多放几件事。

<StudioStep n={1} title="先搭好经典结构">
  打开 `MagicFloor` 里的 Script，确认（没有就补上）这段：

```lua
local part = script.Parent

local function onTouch(hit)
	part.BrickColor = BrickColor.Random()
end

part.Touched:Connect(onTouch)
```
</StudioStep>

<StudioStep n={2} title="往包裹里加“消失”">
  在 `onTouch` 里，变色那行**下面**再加两行，让它被踩到后消失、还能被穿过：

```lua
local part = script.Parent

local function onTouch(hit)
	part.BrickColor = BrickColor.Random()
	part.Transparency = 1
	part.CanCollide = false
end

part.Touched:Connect(onTouch)
```

  - `part.Transparency = 1`：变全透明，看不见了。
  - `part.CanCollide = false`：不再挡人，玩家能直接穿过去。
</StudioStep>

<StudioStep n={3} title="试玩验收">
  打开 **Output**，点 ▶️ **Play**，走上地板：它**先变色、再消失**，你的小人就掉/穿过去了——魔法地板成功！
</StudioStep>

<StudioStep n={4} title="炫耀环节 🎉">

</StudioStep>
