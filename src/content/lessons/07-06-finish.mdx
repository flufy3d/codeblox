## 🎯 今天要做什么

给 Obby 装上**终点**！🏁 玩家冲过终点垫，就会：**得 1 分**、**Output 喊出“你赢了！”**、而且**终点拱门唰地变成金色**——满满的通关仪式感！

## ✨ 新知识：终点垫 = 得分垫 + 胜利效果

好消息：今天的核心你**全学过**。终点其实就是一块特别的**得分垫**（第 6 单元），我们在它上面再加一点点**胜利效果**。

第一步，先有一份**计分板**记“通关次数”。这就是熟悉的 leaderstats（放在 **ServerScriptService**），只是把分数名字叫 `Wins`：

```lua
local Players = game:GetService("Players")

Players.PlayerAdded:Connect(function(player)
	local leaderstats = Instance.new("Folder")
	leaderstats.Name = "leaderstats"   -- 名字必须正好是 leaderstats（全小写）
	leaderstats.Parent = player

	local wins = Instance.new("IntValue")
	wins.Name = "Wins"
	wins.Value = 0
	wins.Parent = leaderstats
end)
```

第二步，**终点垫脚本**（放在终点垫这块 Part 里）：碰到就加分、打印“你赢了！”、再把旁边的拱门变金。我们用熟悉的 **debounce** 防止站着刷分：

```lua
local pad = script.Parent
local arch = workspace.FinishArch   -- 你的终点拱门，名字要和这里一致
local debounce = false

pad.Touched:Connect(function(hit)
	local player = game.Players:GetPlayerFromCharacter(hit.Parent)
	if player and not debounce then
		debounce = true
		player.leaderstats.Wins.Value += 1
		print("你赢了！")
		arch.Material = Enum.Material.Neon
		arch.Color = Color3.fromRGB(255, 215, 0)   -- 金色
		task.wait(2)
		debounce = false
	end
end)
```

<Callout type="info">
  代码里的 `workspace.FinishArch` 是去 Workspace 里**按名字找**你的终点拱门。所以下一步你要放一个拱门 Part，并把它的 **Name 改成正好 `FinishArch`**，名字一字不差，脚本才找得到它。
</Callout>

<StudioStep n={1} title="放好计分板">
  确认 **ServerScriptService** 里有上面第一段计分板脚本（含 `Wins`）。没有就新建一个 **Script** 抄进去。
</StudioStep>

<StudioStep n={2} title="搭一个终点拱门">

</StudioStep>

<StudioStep n={3} title="放终点垫 + 加脚本">
  在拱门下面地上放一块 **Part** 当终点垫，里面 **Insert Object → Script**，整段抄入上面第二段终点垫代码。
</StudioStep>

</StudioStep>
