## 🎯 今天要做什么

学会一个超重要的小魔法：当有东西碰到方块时，找出**到底是哪个玩家**碰的。认出玩家，下一节才能给他**加分**。

## ✨ 新知识：hit 不是玩家，是“碰上来的零件”

还记得 **Touched（碰到）** 事件吗？它会给我们一个 `hit`。但要小心——`hit` **不是玩家本人**，它只是碰上来的**一个身体部件**（可能是脚、手、身子）。

玩家的身体部件，都装在他的**角色模型（Character）**里，也就是 `hit.Parent`。我们用 **Players 服务**的一个本领，把角色换成玩家：

```lua
local player = game.Players:GetPlayerFromCharacter(hit.Parent)
```

如果碰上来的真是个玩家，`player` 就有值；如果是别的方块碰过来，`player` 会是空的，所以我们用 `if player then` 先确认一下：

```lua
local pad = script.Parent

pad.Touched:Connect(function(hit)
	local player = game.Players:GetPlayerFromCharacter(hit.Parent)
	if player then
		print(player.Name)
	end
end)
```

<Callout type="info">
  这个脚本放在 **Pad 这个 Part 里面**（`script.Parent` 就是 Pad 自己），因为它只管这一块垫子被碰到的事。
</Callout>

<StudioStep n={1} title="准备一块 Pad">
  在 **Workspace** 里放一个 **Part**，当作得分垫。**右键这个 Part → Insert Object → Script**。
</StudioStep>

<StudioStep n={2} title="写找玩家的代码">
  把上面那段代码写进去。`script.Parent` 指的就是这块 Pad。
</StudioStep>

<StudioStep n={3} title="试运行">
  按 **▶️ Play**，走上去踩这块 Pad，**Output** 里会打印出你的名字！
</StudioStep>

## 🕹️ 看看你的成果

踩一下，名字就被打印出来——说明游戏认出了“碰到垫子的那个人”是谁。下一节，我们就把“认出玩家”和“**加 1 分**”连起来！

<Callout type="warning">
  没反应？确认脚本放在 **Pad 里面**（不是 ServerScriptService），还有 **GetPlayerFromCharacter** 的大小写要对。
</Callout>
