## 🎯 今天要做什么

把这一单元学的全用上，做一个**真正能玩的小关卡**：一条**不停换色的彩虹跑道**🌈，加上一块**踩到就重来的熔岩**🌋。做完，去炫耀！

## ✨ 新知识：把本领“拼”起来

今天**没有新语法**，全是把学过的拼起来：

- `for` 排出一排跑道格子（第 4 课）
- `math.random` 给每格随机颜色（第 5 课）
- `while` + `task.wait` 让颜色**不停刷新**（第 3、7 课）
- `Touched` + `if` 判断做熔岩重来（第 6 课）

会编程的人，做大东西就是这样——**把小本领一块块拼起来**。

<StudioStep n={1} title="第一段：会变色的彩虹跑道">
  在 **ServerScriptService** 里新建 **Script**，整段抄入：

```lua
local blocks = {}

for i = 1, 10 do
	local block = Instance.new("Part")
	block.Size = Vector3.new(4, 1, 4)
	block.Position = Vector3.new(i * 5, 5, 0)
	block.Anchored = true
	block.Material = Enum.Material.Neon
	block.Parent = workspace
	blocks[i] = block
end

while true do
	for i = 1, 10 do
		blocks[i].Color = Color3.fromRGB(math.random(1, 255), math.random(1, 255), math.random(1, 255))
	end
	task.wait(0.5)
end
```

  先用 `for` 排出 10 个发光格子并记住它们；再用 `while` **一直**重复——每半秒给每个格子换一次随机色。跑道就**活**起来了！
</StudioStep>

<StudioStep n={2} title="试玩跑道">
  点 ▶️ **Play**，看那排格子**不停地变换颜色**，像流动的彩虹。🌈
</StudioStep>

<StudioStep n={3} title="第二段：放一块熔岩">
  ⏹️ Stop。放一个又宽又扁的 **Part**，**Color** 染红、**Anchored** 打勾，摆在跑道旁边当熔岩。
</StudioStep>

<StudioStep n={4} title="给熔岩加判定脚本">
  在 Explorer 里右键这块熔岩 → **Insert Object → Script**，整段抄入：

```lua
local lava = script.Parent

local function onTouch(hit)
	local humanoid = hit.Parent:FindFirstChild("Humanoid")
	if humanoid then
		humanoid.Health = 0
	end
end

lava.Touched:Connect(onTouch)
```

  踩到熔岩的**如果**是个角色（有 Humanoid），**就**让它血量归零、回起点重来。
</StudioStep>

<StudioStep n={5} title="完整试玩你的关卡">
  点 ▶️ **Play**。在闪动的彩虹跑道上走一走，再故意踩一下红色熔岩——小人“重来”了！你的第一个**有跑道、有机关**的小关卡完成啦！🏆
</StudioStep>

<Callout type="tip">
  想要更稳：确认跑道格子和熔岩的 **Anchored** 都打了勾，免得一试玩就掉下去。
</Callout>
