## 🎯 今天要做什么

给 Obby 加一个**会自己来回移动的平台**！↔️ 玩家得**看准时机跳上去**，跟着它过到对岸。这是跑酷里最经典、最刺激的“移动桥”机关。

## ✨ 新知识：用 while 让平台来回动

第 5 单元我们用 **while 循环 + `task.wait`** 让方块不停换颜色。今天用一模一样的本领，只不过这次我们改的不是颜色，而是 **Position（位置）**——方块就会**移动**。

思路很简单，让它**反复做两件事**：

> 移到 A 点 → 等一会 → 移到 B 点 → 等一会 → （再回 A，永远循环）

我们用一个 `startPos` 记住平台的起始位置，再用 `Vector3.new(8, 0, 0)` 表示“往旁边挪 8 格”：

```lua
local platform = script.Parent
local startPos = platform.Position

while true do
	platform.Position = startPos + Vector3.new(8, 0, 0)
	task.wait(1)
	platform.Position = startPos
	task.wait(1)
end
```

读出来就是：**一直**重复——把平台挪到右边 8 格、等 1 秒、再挪回原位、等 1 秒……它就**来回动**起来了。

<Callout type="info">
  这段脚本放在**移动平台这块 Part 里面**（`script.Parent` 就是平台自己）。它只管自己来回动，不影响别的方块。
</Callout>

  在两块固定平台**之间留一段空隙**，放一个 **Part** 当移动桥，**Anchored** 打勾（它靠脚本移动，不靠重力）。
</StudioStep>

<StudioStep n={2} title="给它加脚本">
  在 Explorer 里右键这块平台 → **Insert Object → Script**，整段抄入上面的来回移动代码。
</StudioStep>
