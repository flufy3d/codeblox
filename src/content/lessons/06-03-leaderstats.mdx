## 🎯 今天要做什么

给每个进来的玩家发一张**计分卡**！我们用代码新建一个叫 **leaderstats** 的文件夹，挂到玩家身上——这是 Roblox 显示分数的“官方暗号”。

## ✨ 新知识：用 Instance.new 造一个文件夹

之前都是在 Explorer 里**手动**插入东西。今天学一招更厉害的：用代码**造**东西！

```lua
local leaderstats = Instance.new("Folder")
```

`Instance.new("Folder")` 就是“造一个新文件夹”。造好后要做两件事：给它**取名字**、把它**挂到玩家身上**。

<Callout type="warning">
  名字必须正好是全小写的 **leaderstats**！这是 Roblox 认的“暗号”，写成 `Leaderstats` 或 `LeaderStats` 都不行，计分板就不会出现。
</Callout>

完整片段（接着上节课的 PlayerAdded）：

```lua
local Players = game:GetService("Players")

Players.PlayerAdded:Connect(function(player)
	local leaderstats = Instance.new("Folder")
	leaderstats.Name = "leaderstats"   -- 名字必须正好是 leaderstats（全小写）
	leaderstats.Parent = player
end)
```

<Callout type="info">
  这段还是要放在 **ServerScriptService** 的 Script 里，这样对**所有玩家**都生效。
</Callout>

<StudioStep n={1} title="打开服务器脚本">
  在 **ServerScriptService** 里的 Script，把上面整段代码写进去。
</StudioStep>

<StudioStep n={2} title="看清三步">
  造文件夹 → 取名 `leaderstats` → `Parent = player`（挂到玩家）。三步缺一不可。
</StudioStep>

<StudioStep n={3} title="试运行">
  按 **▶️ Play**，在 **Explorer** 里点开 **Players → 你的名字**，能看到一个 **leaderstats** 文件夹挂在下面。
</StudioStep>

## 🕹️ 看看你的成果

玩家身上多了一个 `leaderstats` 文件夹。它现在是空的——下一节我们往里放一个**分数**，左上角就会出现计分板！

<Callout type="warning">
  Explorer 里看不到文件夹？多半是名字大小写不对，或者 `Parent = player` 那行漏了。
</Callout>
