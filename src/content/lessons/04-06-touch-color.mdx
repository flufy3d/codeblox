## 🎯 今天要做什么

终于到真正的魔法了！把上一课的 `print` 换成**变色**，让你的地板**每碰一次就换一种颜色**。这正是 AUTHORING 里那段经典代码做的事！🌈

## ✨ 新知识：碰到 → 改方块的颜色

还记得我们的“包裹” `onTouch` 吗？现在不打印文字了，改成**给方块换颜色**：

```lua
local part = script.Parent

local function onTouch(hit)
	part.BrickColor = BrickColor.Random()
end

part.Touched:Connect(onTouch)
```

- `part.BrickColor = ...`：改这块地板的颜色。
- `BrickColor.Random()`：**随机**挑一种颜色，所以每次碰到都可能不一样！

如果你想要**固定一种颜色**，把那行换成：

```lua
	part.BrickColor = BrickColor.new("Bright red")
```

把引号里的 `Bright red` 换成别的颜色名，比如 `Bright blue`、`Lime green`、`Hot pink`。

<Callout type="info">
  `BrickColor` 是 Roblox 给颜色起的“名字本”。`Random()` 是随机翻一页；`new("名字")` 是直接翻到你点名的那一页。
</Callout>
