## 🎯 今天要做什么

把这一单元学的**全用上**，写出你的成果作品：一段脚本，让方块**自己变色、自己发光**，还会在 Output 里跟你打招呼。然后——**给家人秀一秀！** 🌈

## ✨ 复习：把命令串成一段脚本

今天不学新东西，把学过的连成一条龙：

> 找到方块 → 改颜色 → 改材质（发光）→ 打印一句话报告

<StudioStep n={1} title="准备方块和脚本">
  在 Explorer 里**右键 Workspace → Insert Object → Part**，改名叫 **NeonBlock**。再**右键 NeonBlock → Insert Object → Script**，把脚本放进它里面。
</StudioStep>

<StudioStep n={2} title="整段写上成果脚本">
  双击脚本，把下面这段**整段抄上去**。注释（`--` 后面的字）帮你记住每行在干嘛：

```lua
local part = script.Parent                       -- 找到我住的那个方块（NeonBlock）
part.BrickColor = BrickColor.new("Lime green")   -- 把颜色改成青柠绿
part.Material = Enum.Material.Neon               -- 改成霓虹材质，让它发光
print("方块准备好啦！")                            -- 在 Output 里报个信
```
</StudioStep>

<StudioStep n={3} title="试玩验收">
  点 ▶️ Play。NeonBlock 应该**变成发光的青柠绿**，同时 Output 里打出 **方块准备好啦！** 全中啦就成功！
</StudioStep>

<StudioStep n={4} title="换成你的专属颜色">
  把 `"Lime green"` 换成你最喜欢的颜色名（如 `"Hot pink"`、`"Bright blue"`），做一个**只属于你**的霓虹方块。
</StudioStep>

<StudioStep n={5} title="炫耀环节 🎉">
  按截图键拍下你发光的方块，或者直接叫爸爸妈妈来——告诉他们：“**这是我用代码做出来的！**”再指着脚本，用自己的话讲讲每一行在干嘛。讲得出来，你就真的懂了。
</StudioStep>

<Callout type="tip">
  完成后别忘了 **Ctrl + S** 保存。给你的作品起个名字吧，比如“我的第一颗发光宝石”——它就正式属于你了。
</Callout>
