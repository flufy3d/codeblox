## 🎯 今天要做什么

上一课你已经偷偷用过**函数**了（就是那段 `onTouch`）。今天我们把它彻底搞懂——函数就是“**把要做的事打包**”，这样代码又整齐又好用。

## ✨ 新知识：函数是“打包好的包裹”

想象你把“碰到地板要做的事”全部装进一个包裹，给包裹贴上名字 `onTouch`。以后只要把这个包裹交给事件，碰到时它就会**自动打开、把里面的事全做一遍**。

```lua
local part = script.Parent

local function onTouch(hit)
	print("有人碰到我啦！")
	print("脚步声～")
end

part.Touched:Connect(onTouch)
```

- `local function onTouch(hit)` … `end`：这中间就是“包裹里的东西”——想做几件事就写几行，这里写了两句 `print`。
- `part.Touched:Connect(onTouch)`：把这个**包裹**交给 Touched 事件。注意是 `onTouch`，**不加括号**——因为我们是把包裹交出去，等碰到了它自己会打开。

<Callout type="tip">
  函数的好处：要做的事很多时，全打包进 `onTouch` 里，最后那行 `Connect` 永远只写一次，特别清楚。
</Callout>
